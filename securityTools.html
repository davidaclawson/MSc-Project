Security Tools

On this page, you will be able to find a variety of security tools that will help yourself, the user, become compliant with the requisites set forth by the hybrid cybersecurity framework used to undertake the gap analysis of your organisation.


These security tools and sources will be categorised by the same categories used in the hybrid cybersecurity framework, as follows:

  • Identify:

    • NCSC: Asset Management
    • European Union: Cybersecurity Policies
    • UK Government: Cybersecurity Responsbilities


  • Protect:

    • NCSC: Encryption & Authentication
    • NCSC: Cybersecurity Training
    • NCSC: Device Management
    • ManageEngine: Endpoint Central (Securing of IT Infrastructure)


  • Detect

    • New Relic: Free end-to-end Monitoring
    • Malwarebytes: Rootkit Scanner
    • ESET Protect Complete: Threat Detection


  • Respond

    • NCSC: Incident Response
    • BetterStack: Incident Response Tool


  • Recover

    • NCSC: Principle Response and Recovery Planning
    • SentinelOne: One Platform SIEM
