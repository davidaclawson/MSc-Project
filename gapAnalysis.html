Gap Analysis Tool

Please find a below a tool to be utilised when undertaking a gap analysis of your own current security strategy. To use it, a user will enter their name or company name, and then will fill each input box below with the level of compliance they are adhering to. This compliance status can be split into three categories:

  • Fully Compliant
  • Partly Compliant
  • Non Compliant
This information will be stored in a back-end database, and users will be able to see their inputted information by heading to the following webpage here: Gap Analysis Data . Users will also be able to delete their inputted information on this webpage by entering their name at the bottom of the page.


Please use the reference ID to refer to the corresponding requisite on the previous webpage to fully understand the requirements of each requisite.
